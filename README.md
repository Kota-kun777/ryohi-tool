# 出張精算ツール

中小企業向けの出張旅費精算ツール。ブラウザ上で動作するシングルHTMLアプリケーション。

## 🌟 特徴

- ✅ 完全ブラウザ上で動作（サーバー不要）
- ✅ スマホ対応（レスポンシブデザイン）
- ✅ ホーム画面追加でアプリ風に使用可能
- ✅ データ自動保存（LocalStorage）
- ✅ JSON形式でデータ保存・読込
- ✅ 印刷・PDF出力対応
- ✅ 仕訳自動生成

## 📋 機能

### 出張区分対応
- 国内日帰り出張
- 国内宿泊出張
- 海外出張（5地域区分）

### 精算方式
- 方式B（規定額-実費の差額支給）
- 日当定額支給
- 業務関連費用（別科目計上）
- 家族同伴補助

### 自動計算
- 規定額 vs 実費の差額
- 個人への非課税支給額
- 会社総支出
- 仕訳例の自動生成

## 🔧 使い方

1. HTMLファイルをブラウザで開く
2. 出張情報を入力
3. 実費を入力
4. 自動計算結果を確認
5. 印刷またはPDF保存

## 📱 スマホで使う

### iPhone (Safari)
1. ページを開く
2. 共有ボタン → 「ホーム画面に追加」
3. アプリのように使用可能

### Android (Chrome)
1. ページを開く
2. メニュー → 「ホーム画面に追加」

## ⚙ 規程額のカスタマイズ

HTMLファイル内の `RATES` オブジェクトを編集することで、自社の規程額に合わせて調整できます。

```javascript
const RATES = {
  officer: {
    flight: { 2: 400000, 3: 700000, 4: 800000, 5: 1300000 },
    hotel: { 1: 30000, 2: 45000, 3: 55000, 4: 70000, 5: 90000 },
    daily: { 1: 15000, 2: 25000, 3: 30000, 4: 30000, 5: 35000 }
  },
  // ...
};
```

## 🔒 プライバシー

- データはブラウザのLocalStorageに保存
- サーバー送信なし
- オフライン動作
